# Softy
Basic software renderer in C.

## Build and run
```bash
$ bash build.sh && ./build/softy
```

## Libraries Used
- [SDL2](https://wiki.libsdl.org/SDL2/FrontPage): creating a window
- [stb](https://github.com/nothings/stb): loading of images and generating font bitmap
